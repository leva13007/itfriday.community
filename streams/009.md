## Про що поговоримо

### Блок 1 — tmux основи + конфіг

- Ієрархія session → window → pane, чому detach/attach — головна фішка tmux
- Розбір власного конфігу через реальні баги: чому виділення мишкою "зникає" і не копіюється в системний clipboard на macOS (і чому це не баг), різниця iTerm2 vs Terminal.app
- Покроковий дебаг tmux-resurrect / tmux-continuum — коли плагіни "начебто встановлені", але нічого не відновлюється після ребута

### Блок 2 — Live demo: кілька AI-агентів паралельно

- Проблема: кілька Claude Code сесій в одній робочій копії — конфлікти файлів
- Рішення: один агент = одна git worktree = одна tmux-панель, кожна підписана
- Наживо: запуск, паралельна робота агентів, фінальний merge гілок назад
