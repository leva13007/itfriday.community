# Публікації

Думки, спостереження і матеріали — без зобов'язань по частоті. З'явилась ідея — написали.

**Три рівні:**
- **Нотатка** — 200–400 слів, думка або спостереження
- **Пост** — 500–800 слів, тема з контекстом
- **Стаття** — 1000+ слів, глибокий матеріал

Теги: `community` — про ком'юніті · `tech` — технічні матеріали · `career` — кар'єра і людське
