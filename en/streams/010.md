## What we'll cover

A tour of real attack scenarios against web applications — from the frontend through the backend to the network.

### Frontend attacks

XSS (Reflected / Stored / DOM-based), CSRF, Clickjacking, CORS misconfiguration, Open Redirect, Reverse Tabnabbing, Prototype Pollution, compromised npm packages (Supply Chain Attack)

### Backend attacks

SQL Injection, Command Injection, Path Traversal / LFI, SSRF, Insecure Deserialization, IDOR / Broken Access Control, Mass Assignment, XXE, Race Condition / TOCTOU

### Network, authentication, infrastructure

Broken Authentication / Session Fixation, Brute Force / Credential Stuffing, Man-in-the-Middle, DDoS, Subdomain Takeover
