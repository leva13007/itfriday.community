# Stream Formats

Every stream belongs to one of 10 directions.

| # | Direction | Guest |
|---|---|---|
| 1 | **Technical deep-dive** — changelog, RFC, TC39 proposals | Optional |
| 2 | **Live coding** — open the editor and try things live | Optional |
| 3 | **Conference review** — React Summit, Next.js Conf, etc. | Optional |
| 4 | **Human / career story** — real story from a guest | Required |
| 5 | **Interview** — structured conversation, questions sent in advance | Required |
| 6 | **Q&A / "Question Time"** | Optional |
| 7 | **IT news digest** — top stories of the week | Optional |
| 8 | **Code review** — anonymous or with the author | Optional |
| 9 | **Interview prep** — mock interviews, question breakdowns, system design | Optional |
| 10 | **Demo** — community projects, tools, startup showcase | Required |
| 11 | **Competitions / Challenges** — Coding Battles, Debugging Challenge, System Design Arena, AI Hack Nights, QA Bug Hunt *(future direction — needs critical mass)* | Required |

## Content plan

Technical and human topics alternate in the schedule.

Host's primary stack: React, Next.js, NestJS, Node.js, TC39, WHATWG.  
Guests can bring any stack.
