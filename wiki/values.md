# Цінності

Це внутрішній компас: коли незрозуміло що робити — відповідь тут.

## 1. Ламповість важливіша за масштаб

Мета — не мільйон переглядів. Мета — 50 людей які справді залучені.
Краще маленький стрім де всі знають одне одного, ніж великий де ніхто нікому не цікавий.

## 2. Технічне і людське — рівноцінні

Розбір нового React API і розповідь про вигорання після переїзду — однаково важливі.
IT — це не тільки код. Це спосіб жити.

## 3. Без перфекціонізму

Стрім без ідеального освітлення — краще ніж відкладений бо треба "ще підготуватись".
Помилка в коді наживо — це не провал, це навчання разом.

## 4. Безпечний простір для "дурних питань"

Немає питань які соромно ставити.
Токсичність і зневага до новачків — не тут.

## 5. Реальність понад маркетингом

Ніяких "топ-5 секретів успіху". Реальні историї, включаючи провали.
Якщо щось не вийшло — краще сказати відкрито.

## 6. Причетність для тих хто далеко

Ком'юніті збирає людей з різних країн.
Еміграція = ізоляція для багатьох. Це місце де можна бути "серед своїх" незалежно де ти.
