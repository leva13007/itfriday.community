# Формати стрімів

Кожен стрім відноситься до одного з 10 напрямків.

| # | Напрямок | Гість |
|---|---|---|
| 1 | **Технічний розбір** — changelog, RFC, TC39 proposals | Optional |
| 2 | **Live coding** — відкриваємо редактор і пробуємо наживо | Optional |
| 3 | **Огляд конференції** — React Summit, Next.js Conf та ін. | Optional |
| 4 | **Людська / кар'єрна тема** — реальна історія гостя | Required |
| 5 | **Інтерв'ю** — структурована розмова, питання надсилаються заздалегідь | Required |
| 6 | **Q&A / "Час Питань"** | Optional |
| 7 | **IT-дайджест** — топ новини тижня | Optional |
| 8 | **Code review** — анонімний або з автором | Optional |
| 9 | **Interview prep** — mock-інтерв'ю, розбір питань, system design | Optional |
| 10 | **Demo** — showcase проектів, інструментів, стартапів | Required |

## Розклад

Контент-план чергує технічні і людські теми.

Основний стек ведучого: React, Next.js, NestJS, Node.js, TC39, WHATWG.  
Гості можуть приносити будь-який стек.
